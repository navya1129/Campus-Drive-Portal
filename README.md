🎓 Campus Job Drive Portal with Resume Match Analyzer

A full-stack web application that helps colleges manage campus placement drives and allows students to explore job opportunities, analyze resume compatibility with job descriptions, and apply for jobs.

This system simplifies communication between students and placement coordinators and provides a centralized portal for campus recruitment.

🚀 Features
👨‍🎓 Student Features

View all available campus job drives

See job details including company, role, salary, and eligibility

Resume Match Analyzer to check compatibility with job description

Apply to jobs directly from the portal

Track applied jobs in My Applications

Secure login using JWT authentication

🧑‍💼 Placement Coordinator Features

Add new job drives

Upload company logos

Manage all job postings

Delete outdated job drives

View job applications from students

🧠 Resume Match Analyzer

Students can paste their resume text and compare it with the job description.

The system calculates:

Match score percentage

Matched skills

Missing skills

This helps students improve their resumes before applying.
